Introduction

Referencer is a Gnome application to organise documents or references, and ultimately generate a BibTeX bibliography file. Referencer includes a number of features to make this process easier:

  • Smart web links
    Referencer uses documents' metadata to provide handy links to the document's web location — no need to maintain your own bookmarks.
  • Import from BibTeX, Reference Manager and EndNote
    No need to start from scratch — Referencer will import your existing bibliography files using the BibUtils library.
  • Tagging
    No need to organise your documents into rigid directory trees — with Referencer you can use tags to categorise your documents.
  • Automatic arXiv, PubMed and CrossRef metadata retrieval
    If you show Referencer a PDF which has an arXiv ID or DOI code, Referencer will retrieve the metadata for this document over the internet.
  • Python plugin support
    Referencer can be extended using the versatile Python scripting language.
  • Localisation
    Referencer is translated into many languages.

Getting Referencer

The latest version of referencer is 1.1.1, the Bring your dog release, 1st March 2008 (ChangeLog)

Source code: referencer-1.1.1.tar.gz

Ubuntu package: Get it from GetDeb

Note: distribution packages may not always correspond to the latest version of referencer. If there is not yet a package for your distribution, you'll need to compile referencer from source. See the development page for more details.

Fedora package: Referencer is available from the fedora-extras repository for Fedora Core versions 6 and 7.

Gentoo Ebuild: Bugzilla thread

Mandriva package:
Mandriva Linux 2008 has Referencer 1.1.1 in its /contrib/backports repository, and Referencer 1.0.4 in its /contrib/release repository. More information is available on installing software and using backports. Older packages are available for earlier releases.

Debian package: Referencer is available in debian unstable

ZenWalk package: Referencer is available in the main ZenWalk repository
