Referencer

A document organiser and bibliography manager for Gnome

Development

Mailing list

Please send questions and suggestions to the mailing list.

For anything which you'd prefer not to discuss in public, you can also contact the main author directly: jcspray at icculus.org

Subversion

To check out the source code:
svn co svn://svn.icculus.org/referencer/trunk referencer

Web view of the subversion repository:http://svn.icculus.org/referencer/

Building

The build dependencies on ubuntu are: build-essential libpoppler-glib-dev libgtkmm-2.4-dev libgnomeuimm-2.6-dev libgnome-vfsmm-2.6-dev libglademm-2.4-dev libgconfmm-2.6-dev libboost-regex-dev intltool.

To work from subversion you'll also need: subversion gnome-common automake-1.9. Note that when building from subversion ./autogen.sh needs to be run before ./configure.

Translation

We're currently trying out launchpad.net for translation. The web interface should allow translation without any specialist knowledge. Please report any problems to the mailing list.
